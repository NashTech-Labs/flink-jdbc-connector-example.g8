Flink JDBC Connector Example

This is an activator project showcasing use of Flink JDBC Connector.

Here we are using:

Flink: for Flink environment.
MySQL: for database connections.

Pre-Requiste:

MySQL must be installed and running.
For this example we need two database source from which we will copy data and sink in which we will copy data. We need data table in source database with two fields id which is integer and name which is a text or a varchar similarly we need a table mydata table in sink database with same fields as data table. 
        CREATE TABLE source.data(id INT, name TEXT);
        INSERT INTO TABLE VALUES (1, "Knoldus"),(2, "Knol");

Run Connector

         bin/activator "run-main com.knoldus.FlinkJDBCExample
This will copy data from one table to another. For any issue please raise a ticket @ Github Issue
